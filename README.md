# <fishing_memory>

## サイト概要
### サイトテーマ
<釣り場所を検索したり、自分が釣った魚などの記録を投稿できるコミュニティサイト>

### テーマを選んだ理由
<魚釣りが好きで、釣れた時の魚の記録と気候の状態や、どんなしかけを使ったかなどを残すサービスがあれば便利だと考え、このテーマにしました。>

### ターゲットユーザ
<魚釣りが好きな人>

### 主な利用シーン
<釣りに行きたい場所、天候や海の状態などを検索するとき、自分の釣れた成果を投稿するとき>
